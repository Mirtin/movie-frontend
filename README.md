## Frontend for Movie App

The Frontend takes data from [Movie Api](https://github.com/Mirtin/movie-api)

## Features Included
- Mui
- Axios
- Custom components and functions
- Cookies
- Pagination

```sh

```
step 2: Run project
```sh
npm start
```
