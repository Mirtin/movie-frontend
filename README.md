## Frontend for Movie App

The Frontend takes data from [Movie Api](https://github.com/Mirtin/movie-api)

## Features Included
- Mui
- Axios
- Custom coponents and functions
- Cookies
- Pagination

## How to run project
step 1: Install libs
```sh
npm install package.json
```
step 2: Run project
```sh
npm start
```
